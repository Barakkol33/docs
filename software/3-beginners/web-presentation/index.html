SKILL 1

VS Code

A supercharged Notepad

  • File Explorer (left sidebar) — browse your files
  • Editor (center) — view and edit files
  • Terminal (bottom panel) — built-in command line
  • Extensions — plugins that add new abilities
  • Command Palette — Ctrl+Shift+P to find any action

SKILL 2

Command Line

A text-based way to talk to your computer

Bash (Mac / Linux)

The default on Mac and Linux. Most tutorials assume Bash.

CMD & PowerShell (Windows)

CMD is basic. PowerShell is more powerful and supports similar commands to Bash.

SKILL 4

Python

A beginner-friendly programming language

# Run from terminal
python my_script.py

# Install a module
pip install pandas

# Use it in code
import pandas

Got ModuleNotFoundError? Just pip install it.

SKILL 5

HTML, CSS & JavaScript

The building blocks of websites

  • HTML — structure & content
  • CSS — colors, fonts, layout
  • JavaScript — interactivity & behavior

Just open any .html file in your browser to see it work.

SKILL 6

Claude Code

An AI assistant that edits files for you

  • Edit files — "change the heading color to red"
  • Create files — "make a website with a contact form"
  • Explain code — "what does this file do?"
  • Run commands — it runs terminal commands on your behalf
